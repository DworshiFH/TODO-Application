Dworscak Andreas WP1/BA1 Projekt

Wilkommen beim WP1 von Andreas Dworscak. Eine Todo-Application mit dem Zweck die Funktionen des PyScript Frameworks zu untersuchen.

Zu diesem Zweck wurden mehrere Versionen, jeweils mittels JavaScript und PyScript, dieser Application erstellt. Ein PHP-basiertes Backend dient als Schnittstelle zu einer serverseitigen Datenbank.
